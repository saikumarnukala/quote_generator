## 📁 Getting Started

### Prerequisites

- A modern web browser

### Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/saikumarnukala/quote_generator.git
   ```

2. Navigate to the project folder:

   ```bash
   cd quote_generator
   ```

3. Open `index.html` in your browser.
